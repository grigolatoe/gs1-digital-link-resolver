# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its
Code of Conduct. The full text — including standards, enforcement
responsibilities, scope, and enforcement guidelines — is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

By participating in this project (issues, pull requests, discussions,
or any other project space), you agree to abide by that document.

## Reporting

If you experience or witness conduct that violates the Code of Conduct,
please email **conduct@grigolato.it**.

Reports are handled promptly and confidentially. Reporters may remain
anonymous, and we will not disclose a reporter's identity in any public
communication about an incident without their explicit consent.
